pub enum SdkError<E> {
    ConstructionFailure(Box<dyn Error + Sync + Send + 'static, Global>),
    DispatchFailure(Box<dyn Error + Sync + Send + 'static, Global>),
    ResponseError {
        raw: Response,
        err: Box<dyn Error + Sync + Send + 'static, Global>,
    },
    ServiceError {
        err: E,
        raw: Response,
    },
}
Expand description

Failed SDK Result

Variants

ConstructionFailure(Box<dyn Error + Sync + Send + 'static, Global>)

The request failed during construction. It was not dispatched over the network.

DispatchFailure(Box<dyn Error + Sync + Send + 'static, Global>)

The request failed during dispatch. An HTTP response was not received. The request MAY have been sent.

ResponseError

A response was received but it was not parseable according the the protocol (for example the server hung up while the body was being read)
